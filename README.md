# Grand Canyon carving in Fortran
Numerical carving of the Grand Canyon in Fortran:
-diffusion equation
-Laplace operator
-2D erosion


## Quick start
```bash
git clone git@github.com:mthpvg/grep2vim.git
sudo apt-get install gfortran
./compileAndExecute
```
